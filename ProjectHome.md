CCNI is a network data transfer protocol for Chinese Chess Programs. It provids a standard way for them to find opponent, play chinese chess game on the internet.<p>
Detailed information about CCNI please refer to:<br>
<a href='http://ccniserver.googlecode.com/svn/branches/ccniserver/docs/CCNIProtolcolWhitePaper.pdf'>http://ccniserver.googlecode.com/svn/branches/ccniserver/docs/CCNIProtolcolWhitePaper.pdf</a><p>


ccniserver is a server side implementation of CCNI which were writen in c++. It will run under linux system and has following features:<br>
<br>
<ul><li><b>Fully support of CCNI.</li></ul>

<ul><li>Game session support.</li></ul>

<ul><li>Hight performance and large scale of simultaneous access.</li></ul>

<ul><li>Complete logging  mechanism.</li></ul>

<ul><li>Simple User database management.</li></ul>

<ul><li>32 bit & 64 bit support.</li></ul>

<ul><li>Easy to porting to different linux distributions.</b></li></ul>
